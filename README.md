# ICPR 2026 Tutorial Website

Tutorial: Advancing Comprehensive Reasoning in Multimodal Large Language Models

## How to use

1. Replace the placeholder images in `images/` with real photos.
2. Keep these filenames:
   - `yiwei_wang.jpg`
   - `yujun_cai.jpg`
   - `junsong_yuan.jpg`
   - `jun_liu.jpg`
   - `ziquan_liu.jpg`
   - `yirui_wu.jpg`
   - `siyuan_yang.jpg`
3. Update the schedule in `index.html` when the final tutorial time is confirmed.
4. Upload the folder to GitHub Pages, Netlify, or any static web hosting service.

## Suggested image format

Use square or portrait photos, preferably at least 500 × 500 px. The CSS will automatically crop them into circular headshots.
